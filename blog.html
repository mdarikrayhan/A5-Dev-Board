Question-1: What are the different ways to select an element in the DOM?

There are five different ways to select an element in the DOM.
Like -->
1. using getElementById().
2. using getElementsByClassName()
3. using getElementsByTagName()
4. using querySelector()
5. and Finally using querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference among innerHTML, innerText and textContent is

1. innerHTML:
innerHTML returns both HTML murkup and text content of an element.

2. innerText:
innerText only returns text content of an element without html tag and the text content is what we see on screen.

3. textContent:
textContent only returns text content of an element without html tag but the text content is what is written in hard code.
It means all disappear's content is also returned

Question-3: What is event delegation in the DOM?

Event delegation is the process to stop spreading the event upwards from the targeted event
and also stop spreading around the targeted event.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process of spreading the event upwards from the targeted event.

Question-5: How do you create, add, and remove elements using JavaScript?

1. using document.createElement(), we can create any kinds of html element.

2. using append, appendChild and so on methods, we can add any kinds of html element inside or outside in others element.

3. using removeChild method, we can delete any kinds of html element.
